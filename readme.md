# Authenticated To-Do
## The Challenge
Build an authenticated To-DO list (independent for each user) using supabase as backend and MUI as component library.

## Built with
- React
- MUI (Component library)
- React Router
- Gravatar 
- Supabase (Backend as a service )
- Formik
- Yup (Object Schema validation)
- Typescript
- TS-Standard
- Vite
